CMPS 311 - Operating Systems

Homework 4

For this assignment, you will write three separate programs that do the exact same thing - which is similar to Homework 3!.

Program 1:        POSIX (using the pthread library) written in C (NOT C++).

Program 2:        Win32 (using the Win32 Thread API) written in C (Not C++).

Program 3:        C++ Threads (C++ 11)

For each of these programs, you will construct a program that repeatedly asks the user for an integer (N), and prints out the first N fibonacci numbers to a file named fib-N.txt (where N is the number the user typed in).

Each time a user types a number (except -1, which will indicate the end of the program), you should spawn worker a new thread (using POSIX, Win32, or C++ depending on which program you are creating) and send in N as the thread parameter.

Your “worker” thread should compute the N fibonacci numbers and save them to the file.  Your main thread/process should not wait for the work thread to complete - theoretically allowing multiple worker threads to be active simultaneously.

Note:  If you are quick enough, you could have multiple workers creating the same text file if you type the same N value in twice in a row.  This is a big problem, and we’ll figure out how to prevent this later - for now pretend this won’t happen :)

Note 2:  I’m serious about you using C for Programs 1 and 2 - you can’t use C++ I/O!

Each of the three programs are work 33.33% of your grade for this homework.
